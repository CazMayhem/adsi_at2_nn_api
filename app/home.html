Welcome To The Bestest Beer Predictor!

This web app demonstrate a machine learning model - to will predict any of 104 different types of beers - based on ratings entered for the following categories:

  1. Brewery name - 5742 available breweries
  2. Aroma - 1 to 5
  3. Appearance - 0 to 5
  4. Palate - 1 to 5
  5. Taste - 1 to 5
  6. Alcohol by volume

The machine learning model was developed using custom neural networks with PyTorch.

API endpoints available by this web app:

1. http://pure-shelf-34977.herokuapp.com/

  • Inputs: none
  • Output: information page

2. http://pure-shelf-34977.herokuapp.com/health/

  • Inputs: none
  • Output: daily health tips

Tip: for a bit of fun, press <cmd-R> (Mac) or <ctrl-R> (Windows) to refresh the page multiple times :)

3. http://pure-shelf-34977.herokuapp.com/beer/type/

  • Inputs:  6 criteria above - brewery name, aroma, etc...
  • Output: One predicted beer style

  • Example API call: 

    https://pure-shelf-34977.herokuapp.com/beer/type?brewery_name=Crown%20Brewing&review_aroma=6&review_appearance=5&review_palate=4.5&review_taste=3.5&beer_abv=1.5

 4. http://pure-shelf-34977.herokuapp.com/beers/type/

  • Inputs:  Multiple instances of the 6 criteria above
    •             - brewery name, aroma, etc...
  • Output: Multiple predicted beer styles

  • Example API call:

    https://pure-shelf-34977.herokuapp.com/beers/type?beer_input=Stone%20Brewing%20Co.%2C1.1%2C2.2%2C3.3%2C4.4%2C5.5&beer_input=Crown%20Brewing%2C1.1%2C1.2%2C1.3%2C1.4%2C1.5
